# WeatherForcast

A simple weather forecast app that shows current weather data using an API.

## Features
- Get real-time weather updates
- Search by city name
- Responsive design

## Technologies
- HTML/CSS/JS (or React, if you're using it)
- Weather API (e.g., OpenWeatherMap)

## Setup
1. Clone the repo
2. Install dependencies
3. Run the project
